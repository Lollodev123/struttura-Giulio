# 📖 Glossario Aeronautico

Termini italiani con equivalente inglese (usato nei manuali) e definizione operativa.

| Italiano | Inglese | Definizione breve |
|---|---|---|
| **Profilo alare** | Airfoil | Sezione trasversale dell'ala |
| **Corda** | Chord | Segmento da bordo d'attacco a bordo d'uscita |
| **Bordo d'attacco** | Leading edge | Bordo anteriore del profilo |
| **Bordo d'uscita** | Trailing edge | Bordo posteriore (tagliente) |
| **Linea media** | Mean camber line | Linea equidistante tra dorso e ventre |
| **Curvatura / freccia max** | Camber | Distanza max tra linea media e corda |
| **Spessore** | Thickness | Distanza tra dorso e ventre |
| **Dorso** | Upper surface | Superficie superiore del profilo |
| **Ventre** | Lower surface | Superficie inferiore del profilo |
| **Angolo di attacco** | Angle of attack ($\alpha$) | Angolo tra corda e velocità asintotica |
| **Angolo di portanza nulla** | Zero-lift angle ($\alpha_0$) | $\alpha$ al quale $C_p = 0$ |
| **Stallo** | Stall | Separazione del flusso che azzera la portanza |
| **Portanza** | Lift | Forza perpendicolare alla velocità |
| **Resistenza** | Drag | Forza parallela alla velocità (opposta al moto) |
| **Spinta** | Thrust | Forza propulsiva del motore |
| **Peso** | Weight | $Q = m \cdot g$ |
| **Sostentazione** | Sustentation | Spesso sinonimo di portanza in italiano (attenzione) |
| **Coefficiente di portanza** | Lift coefficient ($C_p$) | Adimensionale, dipende da profilo e $\alpha$ |
| **Coefficiente di resistenza** | Drag coefficient ($C_R$) | Adimensionale, totale (profilo + indotta) |
| **Polare** | Drag polar | Grafico $C_p$ vs $C_R$ |
| **Efficienza** | Lift-to-drag ratio | $E = C_p/C_R$ |
| **Allungamento alare** | Aspect ratio ($\lambda$ o $AR$) | $b^2/S$ |
| **Apertura alare** | Wingspan ($b$) | Lunghezza dell'ala da tip a tip |
| **Superficie alare** | Wing area ($S$) | Area in pianta dell'ala |
| **Rastremazione** | Taper ratio | $c_{tip}/c_{root}$ |
| **Freccia (alare)** | Sweep | Angolo di freccia dell'ala in pianta |
| **Diedro** | Dihedral | Angolo tra ala e piano orizzontale |
| **Ipersostentatore** | High-lift device | Generico: flap, slat, slot |
| **Flap** | Flap | Superficie mobile sul bordo d'uscita |
| **Slat** | Slat | Superficie mobile sul bordo d'attacco |
| **Slot** | Slot | Fessura fissa che ritarda lo stallo |
| **Strato limite** | Boundary layer | Sottile strato di fluido viscoso vicino alla parete |
| **Numero di Reynolds** | Reynolds number ($Re$) | Rapporto forze inerziali/viscose |
| **Velocità di stallo** | Stall speed ($V_S$) | Velocità minima per volo livellato |
| **Centro di pressione** | Center of pressure | Punto di applicazione della risultante aerodinamica |
| **Fuoco aerodinamico** | Aerodynamic center | Punto dove $C_M$ non dipende da $\alpha$ |
| **Coefficiente di momento** | Moment coefficient ($C_M$) | Misura il momento di beccheggio |
| **Atmosfera standard** | Standard atmosphere (ISA) | Modello di riferimento di $\rho$, $p$, $T$ vs quota |
| **Densità relativa** | Density ratio ($\sigma$) | $\rho/\rho_0$ |
| **Quota di tangenza** | Service ceiling | Quota max raggiungibile dal velivolo |
| **Crociera** | Cruise | Fase di volo livellato a velocità costante |
| **Planata** | Glide | Discesa senza spinta |
| **Stallo profondo** | Deep stall | Stallo non recuperabile (config. T-tail) |
