# Come contribuire

## Per lo studente
- Apri Issue per ogni dubbio (è uno dei modi più efficaci di studiare).
- Se trovi un errore di calcolo o un refuso, apri Issue con il template "Errore nel materiale".

## Per chi cura il repo
- Segui il `_template-lezione.md` e `_template-esercizio.md` per mantenere lo stile uniforme.
- Ogni nuovo esercizio deve avere: testo, diagramma, dati, strategia, risoluzione passo-passo, verifica plausibilità, variante.
- Aggiorna sempre il README con il nuovo link nell'indice.
